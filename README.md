# Joe DV Large Outbreak — Public Figures

Output figures from the Tainan dengue SEIR analysis project. **Figures only**; no raw data, no source code, no reports. Source repo (private): `swhunag65221-pixel/joe_dv_large_outbreak`.

Hosted publicly so they can be embedded in [Notion reports](https://www.notion.so/3428b22c-430e-80da-8bdb-c82581dc47e8) via `raw.githubusercontent.com` URLs.

## Contents

| folder | content | companion Notion / docs |
|--------|---------|--------------------------|
| `district_risk/` | 37-district climate-suitability heatmap | "分區風險地圖" (district risk heatmap) |
| `intervention_shift/` | TN 2023 intervention-timing counterfactual | "介入反事實" (intervention shift) |
| `null_validation/` | Model failure on 2024-25 null-outbreak validation | "Null-outbreak validation" |
| `realtime_rt/` | Real-time walk-forward per-district Rt (2023) | "即時 Rt walk-forward" |
